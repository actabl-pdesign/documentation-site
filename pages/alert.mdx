## Properties

| Property | Description | Values | Default |
| --- | --- | --- | --- |
| variant | Alert type | info, success, warning, error | info |
| dismissible | Can be dismissed | true, false | false |
| title | Alert title | string | - |
| description | Alert description | string | - |
| icon | Show icon | true, false | true |

## Best Practices

- Use appropriate alert types for different message types
- Keep alert messages concise and actionable
- Position alerts where users expect to see them
- Consider auto-dismissing non-critical alerts
- Ensure alerts are accessible to screen readers
