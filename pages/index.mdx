# Bellhop Design System - v0.dev Integration Guide

This comprehensive guide helps developers and designers use the Bellhop Design System effectively with v0.dev for rapid prototyping and component generation.

## Quick Start for v0.dev

### Essential Information for v0.dev Prompts

When working with v0.dev, always include these key details in your prompts:

**Design System Context:**
\`\`\`
Using Bellhop Design System with:
- Primary color: Indigo (#2642CF)
- Secondary color: Pink (#F63D68)
- Typography: Inter font family
- Border radius: 8px standard, 12px for cards
- Spacing: 8px, 16px, 24px increments
\`\`\`

**Quick Setup:**
\`\`\`css
/* Include in your v0.dev prompts for proper styling */
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap');

/* Use these CSS variables */
:root {
  --color-primary: 38 66 207; /* Indigo-600 */
  --color-secondary: 246 61 104; /* Pink-500 */
  --color-gray-50: 240 20% 98%;
  --color-gray-900: 233 33% 10%;
}
\`\`\`
